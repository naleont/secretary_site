Поиск работы

Перейдите: Личный кабинет - Поиск работ или Администрирование - Поиск работ. Выберите, по чему будете искать работу: доступны варианты по номеру, названию, автору, руководителю, городу, организации, e-mail или ID пользователя. Кроме того, можно выставить фильтр по региональному туру.

Введите данные для поиска и нажмите Искать.

Обратите внимание: если вы искали работу по номеру, поиск выдаст все работы с таким номером за все годы. Например, при поиске по номеру 180150 сверху вниз будут отображаться работы 200150, 190150, 180150, 170150 и т.д.

Редактирование работы

Найдите работу. Для изменения информации о работе нажмите значок-карандаш напротив нужной работы. Будьте внимательны: не внесите лишних правок. Кроме информации о работе, вы можете изменить её файлы. Для этого перейдите на вкладку Файлы работы.

После редактирования информации не забудьте нажать кнопку Сохранить.

Поиск пользователя

Перейдите в раздел Администрирование – Пользователи.

Перед поиском нужно выбрать из раскрывающегося списка, по чему будет производиться поиск.

Попробуйте найти пользователя по e-mail, с которого пришло письмо, или по фамилии. Если это не удалось, запросите у пользователя логин, который он придумал при регистрации на сайте, и выполните поиск по нему.

Редактирование пользователя

Найдите пользователя. Нажмите на значок-«визитку», чтобы перейти в режим редактирования контакта. Будьте внимательны: не отредактируйте какое-нибудь лишнее поле.

После редактирования информации не забудьте нажать кнопку Сохранить
